## 💡 实用组合

### 快速开发布局

```
Alt + \       # 右边开编辑器
Alt + h       # 回到左边
Alt + -       # 下方开终端
```

### 查看日志

```
Alt + s       # 滚动模式
/             # 搜索 "error"
n             # 下一个
q             # 退出
```

### 多标签切换

```
Alt + t       # 新建标签
Alt + 1       # 第 1 个标签
Alt + 2       # 第 2 个标签
Alt + 3       # 第 3 个标签
```
